## 写稿规则

| 规则 | 说明 |
|------|------|
| 标题 ≤20字 | 中文一个字算一字，英文一个单词算一字 |
| 图片 1-9张 | 放在 `assets/` 目录，不支持网络图片 |
| tags 必填 | 话题标签，发布时**不会**被平台自动补词条 |
| status | `draft` → 写好改 `ready` → 酱酱发布后自动变 `published` |
| 正文不以 # 开头 | 小红书会把行首 `#xxx` 当话题标签 |

## 正文禁止

- ❌ 文中用 `#话题` 格式（用 tags 字段代替）
- ❌ 诱导点赞/收藏/关注话术
- ❌ 违反社区规范的内容
